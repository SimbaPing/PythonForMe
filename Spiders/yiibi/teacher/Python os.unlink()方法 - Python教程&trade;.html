Python os.unlink()方法

¥ 我要打赏 Python教程 作者：初生不惑 评论：0 条 Java技术QQ群：227270512

Python的unlink()方法删除文件路径。如果路径是目录，则会引发OSError。 功能与remove()函数相同; unlink名称是其传统的Unix名称。

语法

以下是unlink()方法的语法 -

os.unlink(path)

参数

  • path - 这是要删除的路径。

返回值

  • 此方法不返回任何值。

示例

以下示例显示了unlink()方法的用法。

# !/usr/bin/python3
import os, sys
os.chdir("d:\\tmp")

# listing directories
print ("The dir is: %s" %os.listdir(os.getcwd()))
os.unlink("foo.txt")

# listing directories after removing path
print ("The dir after removal of path : %s" %os.listdir(os.getcwd()))

当运行上述程序时，得到如下所示： -

The dir is: [
   'Applicationdocs.docx', 'book.zip', 'foo.txt', 
   'Java Multiple Inheritance.html', 'Java Multiple Inheritance_files', 
   'java.ppt', 'python2'
]

The dir after removal of path : [
   'Applicationdocs.docx', 'book.zip', 
   'Java Multiple Inheritance.html', 
   'Java Multiple Inheritance_files', 'java.ppt', 'python2'
]
